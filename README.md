### 实现功能
* 用微信自带的setStorage和getStorage实现本地存储数据。
* 页面数据是静态的。用wx:for对数据进行循环渲染。
* 添加商品到购物车
* 添加地址(使用微信自带的省市区三栏地址选择)
* 购物车单选，全选，价格计算
